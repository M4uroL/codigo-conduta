# 📜 Código de Conduta — Comunidade Java ☕

*"voluntario == Volunteer"*

---

## 🎯 Nossa Missão

Somos uma comunidade Java aberta, inclusiva e colaborativa. Nosso objetivo é simples: **aprender juntos, compartilhar conhecimento e crescer como profissionais**, num ambiente onde o bom humor e o respeito caminham lado a lado.

Aqui, todo mundo ensina e todo mundo aprende — do primeiro `Hello World` ao microsserviço mais sofisticado.

---

## 🌟 Nossos Valores

| Valor | O que significa na prática |
|-------|-----------------------------|
| **Respeito** | Trate os outros como gostaria de ser tratado. Não importa o nível técnico, tempo de experiência ou ferramenta preferida. |
| **Colaboração** | Ajudar é mais importante que estar certo. A vitória é coletiva. |
| **Curiosidade** | Perguntar é um superpoder. Não existe pergunta besta, existe resposta que ainda não veio. |
| **Humildade** | Ninguém sabe tudo. Reconhecer isso é o primeiro passo para evoluir. |
| **Transparência** | Decisões, mudanças e critérios são comunicados de forma clara para todos. |

---

## ⛔ Comportamentos Inaceitáveis

**Isso não será tolerado em hipótese alguma:**

- Assédio, discriminação ou qualquer forma de preconceito (gênero, raça, orientação, religião, deficiência, etc.).
- Ataques pessoais, xingamentos ou provocações maliciosas.
- Críticas destrutivas — foque no código ou na ideia, nunca na pessoa.
- Spam, autopromoção excessiva ou divulgação fora dos canais adequados.
- Compartilhamento de informações privadas sem consentimento.
- Trollagem, "flame wars" ou comportamento que atrapalhe discussões produtivas.
- Uso de linguagem ou imagens ofensivas, mesmo "em tom de brincadeira".

## 📢 Regras para Anúncios

Se for postar no grupo de anúncios, siga estas regras:

### 💼 Vagas e Estágios
- Informe requisitos, local (ou remoto), benefícios e contato.
- Evite "salário a combinar" — seja transparente sempre que possível.

### 🌐 Projetos Open Source
- Descreva o propósito, stack utilizada, nível de dificuldade e como contribuir.
- Links úteis (repo, issues boas para iniciantes, canal de comunicação).

### 🎫 Eventos e Bootcamps
- Data, formato (online/presencial), link oficial e público-alvo.

📌 **Importante:** anúncios fora dessas diretrizes ou com autopromoção excessiva serão removidos sem aviso prévio.

---

## 🛡️ Como Reportar Problemas

Se você vivenciou ou testemunhou uma violação do código de conduta:

1. Entre em contato com qualquer administrador (grupo `this.admin = admin`).
2. Sua denúncia será tratada com **sigilo total, imparcialidade e celeridade**.
3. Medidas cabíveis podem incluir:
   - Advertência privada
   - Remoção de mensagens
   - Suspensão temporária
   - Banimento permanente (em casos graves ou reincidência)

Nossa política é **zero tolerância** para comportamentos abusivos.
